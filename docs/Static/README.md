## Static

## 1. When naming layouts, first should be it's usage (activity_*, dialog_*, fragment_*, item_*, layout_*, view_*) folowing by screen purpose in app.

Example:
	activity_main
	activity_authentication
	dialog_name_surname
	dialog_date
	fragment_register
	fragment_login
	item_task
	item_setting
	layout_pin
	layout_toolbar
	view_user
	view_rating
